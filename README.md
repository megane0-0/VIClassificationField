# 視野検査 最大直径計算ツール

視覚障害者スポーツのクラス分け判定支援Webアプリケーション

![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 概要

このリポジトリには、視野検査の測定点をインタラクティブに入力し、固視点を通る視野の最大直径を自動計算する2つの独立したWebアプリケーションが含まれています。IBTA（International Blind Tennis Association）などの視覚障害者スポーツにおけるクラス分け判定を支援します。

## 📦 2つのツール

### 1. 🎯 中心視野検査ツール (`central.html`)

ハンフリー視野検査（Humphrey Visual Field）の中心視野測定に対応。

**対応検査タイプ:**
- **ハンフリー 10-2**: 68測定点、2度間隔、中心10度範囲
- **ハンフリー 30-2**: 76測定点、6度間隔、中心30度範囲

**特徴:**
- 正方形領域ベースの視野計算
- リアルタイムでの最大直径計算
- 半盲境界線（0°, 90°, 180°, 270°）の自動除外
- インタラクティブな測定点入力

### 2. 🌐 周辺視野検査ツール (`index.html`)

FF120およびエスターマン視野検査の周辺視野測定に対応。

**対応検査タイプ:**
- **両眼開放エスターマン**: 120測定点、両眼開放での視野測定
- **FF120 右眼 (OD)**: 120測定点、右眼単眼での視野測定
- **FF120 左眼 (OS)**: 120測定点、左眼単眼での視野測定

**特徴:**
- 最近傍補間による境界点計算
- リアルタイムでの最大直径計算
- 半盲境界線（0°, 90°, 180°, 270°）の自動除外
- 検査タイプのワンクリック切り替え

## 📋 共通機能

両ツールともに以下の機能を備えています：

### ✨ 主要機能
- **インタラクティブ入力**: クリックで測定点のオン/オフを切り替え

- **画像保存**: 視野図をPNG画像としてダウンロード
- **一括操作**: 全選択/全解除/リセットボタン

## 📐 計算アルゴリズム

### 中心視野ツール（ハンフリー）
1. オンの測定点を中心に正方形領域を作成（10-2は±1度、30-2は±3度）
2. すべての正方形をマージして連続領域を生成
3. 固視点を通る直線を0.25度刻みで生成（半盲境界線を除外）
4. 各直線と視野領域の交点を計算し、最大直径を特定

### 周辺視野ツール（FF120/Esterman）
1. 各方向で最近傍補間により境界点を計算
2. 境界点から閉じたポリゴンを生成
3. 固視点を通る直線を0.5度刻みで生成（半盲境界線を除外）
4. 各直線と境界ポリゴンの交点を計算し、最大直径を特定

## 🔧 半盲境界線除外機能

両ツールとも、半盲（hemianopia）の境界線となる以下の方向を自動的に除外します：
- **0° (水平右)** と **180° (水平左)**
- **90° (垂直上)** と **270° (垂直下)**

この機能により、半盲の患者における視野直径の過大評価を防ぎます。

## 📝 使い方

### 基本操作（両ツール共通）

1. **検査タイプの選択**
   - 中心視野: 10-2 または 30-2 を選択
   - 周辺視野: エスターマン、FF120右眼、FF120左眼 から選択

2. **測定点の入力**
   - 視野図上の点をクリックして見える/見えないを切り替え
   - 緑色 = 見える、グレー = 見えない

3. **一括操作**
   - 「全選択」: すべての測定点をオンにする
   - 「全解除」: すべての測定点をオフにする
   - 「リセット」: 初期状態に戻す

4. **結果の確認**
   - 最大直径、方向、端点座標が自動的に表示
   - 視野図に最大直径ラインが赤色で表示される

5. **結果の保存**
   - 「結果をコピー」: テキストをクリップボードにコピー
   - 「画像として保存」: PNG形式で視野図を保存

## 🔬 開発者向け情報

### 中心視野ツールのカスタマイズ

**測定点の追加・変更**
\`central-data.js\` の \`HUMPHREY_10_2_POINTS\` または \`HUMPHREY_30_2_POINTS\` を編集

**計算パラメータの調整**
\`central-calculator.js\` の \`calculateMaxDiameter()\` 関数内:
- 角度刻み（デフォルト0.25度）
- 正方形領域のサイズ

### 周辺視野ツールのカスタマイズ

**FF120座標データの置き換え**
\`ff120-coordinates.js\` の \`FF120_RIGHT_EYE_COORDINATES\` 配列を実際のデータで置き換え

**計算パラメータの調整**
\`ff120-calculator.js\` の関連関数:
- 角度刻み（デフォルト0.5度）
- 最近傍補間の距離閾値
